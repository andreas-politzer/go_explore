# Local Data Stack — Setup Guide

A practical guide to building a local, open-source data analysis environment on your Mac — independent of any cloud service — with an optional connection to Google BigQuery for team collaboration.

Built by The Lobster Lookers during the WBS Data Science & AI Bootcamp, Berlin 2026.

## What this covers

- **DuckDB in VS Code** — write and run SQL directly on local files, no server needed
- **Metabase via Docker** — visual dashboards that run entirely on your machine
- **BigQuery connection** — optional bridge to shared team projects in Google Cloud

## Who this is for

Anyone who wants to work with data without depending on Google Sheets, proprietary tools, or expensive cloud subscriptions. The guide assumes no prior experience with any of these tools.
